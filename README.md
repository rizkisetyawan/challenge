# FULLSTACK TECHNICAL CHALLENGE

## Using an open source SQL database using MySQL, create a tablecalled users, with the
following columns :

- id: number
- email: string
- first_name: string
- last_name: string
- avatar: string
- created_at: date
- updated_at: date
- deleted_at: date
  
```bash
  CREATE TABLE users (
    id INT PRIMARY KEY,
    email VARCHAR(100),
    first_name VARCHAR(50),
    last_name VARCHAR(50),
    avatar VARCHAR(100),
    created_at DATE,
    updated_at DATE,
    deleted_at DATE
  );
```
